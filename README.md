# task11_3
Функция get_r2_rmse(), которая принимает две переменные – название csv файла и название целевой переменной. Функция:

Разделяет датасет на тренировочные и тестовые данные в соотношении 3 к 1
Обучает модель на полученной выборке
Рассчитывает параметры R2 и RMSE для полученной модели
Функция возвращает список с найденными параметрами [r2, rmse].
